# Strategickhaos-pipe-trades-cli

Field-calibrated pipefitter calculation ecosystem. GPS coordinate verification, beam wrap material estimation, rolling offset calculations. Mirrors Pipe Trades Pro 4095 + TI-nspire CX II programmability. Built for rope access crews doing fireproofing containment.

## New: Feature Engine Architecture

**Vision → Computation**: A three-layer Excel-DNA system that transforms abstract concepts into measurable outputs.

- **Interface Layer**: Excel cells expose primitive feature functions
- **Orchestration Layer**: C# functions normalize and dispatch
- **Engine Layer**: Pure functions return metrics (no interpretation)

### Available Feature Functions

- `FREQUENCY_HZ(wavelength_mm)` - Physics: wavelength → frequency
- `ELEMENT_Z(symbol)` - Chemistry: element symbol → atomic number
- `GEO_DISTANCE_KM(lat1, lon1, lat2, lon2)` - Geometry: GPS distance
- `FALLACY_SCORE(text)` - Text analysis: detect logical fallacies
- `ENTROPY_SCORE(text)` - Text analysis: Shannon entropy
- `CONTRADICTION_SCORE(text)` - Text analysis: detect contradictions
- `SIGNAL_SHARPNESS(text)` - Text analysis: measure clarity
- `UTRD(text)` - Composite: Universal Truth Reliability Detector
